# Task Manager CLI

A simple command-line task manager built with Node.js and the native `fs` module.

## Features

- ✅ Add a new task
- 📋 List all tasks
- ❌ Remove a specific task
- 🧹 Clear all tasks

## Built With

- Node.js
- Native `fs` module (file system)

## Installation

```bash
git clone <your-repo-url>
cd task-manager-cli
